### Step 1: Install Prerequisites


### Step 2: Create GitHub Repository

1. Go to [GitHub.com](https://github.com) and sign in
2. Click **"New repository"** (green button)
3. Repository name: `hidden-profit-finder`
4. Description: `AI-Powered Hidden Profit Finder - Strategy Simulation Tool`
5. Set to **Public** (required for free GitHub Pages)
6. âœ… Check **"Add a README file"**
7. Click **"Create repository"**

### Step 3: Clone and Setup Project

```bash
# Clone the repository
git clone https://github.com/YOURUSERNAME/hidden-profit-finder.git
cd hidden-profit-finder

# Copy all project files to this directory
# (Copy the contents of each artifact file to the corresponding location)

# Install dependencies
npm install
```

### Step 4: Update Configuration

1. **Edit `package.json`**: Replace `"homepage"` with your GitHub Pages URL:
   ```json
   "homepage": "https://YOURUSERNAME.github.io/hidden-profit-finder"
   ```

2. **Test locally**:
   ```bash
   npm start
   ```
   Should open at `http://localhost:3000`

### Step 5: Deploy to GitHub Pages

```bash
# Add, commit and push changes
git add .
git commit -m "Initial commit: Hidden Profit Finder simulator"
git push origin main

# Deploy to GitHub Pages
npm run deploy
```

### Step 6: Enable GitHub Pages

1. Go to your repository on GitHub
2. Click **Settings** tab
3. Scroll to **Pages** section (left sidebar)
4. Source: **Deploy from a branch**
5. Branch: **gh-pages**
6. Folder: **/ (root)**
7. Click **Save**

**Your site will be live at**: `https://YOURUSERNAME.github.io/hidden-profit-finder`

## ðŸš¨ Common Issues & Solutions

### Issue: npm install fails
```bash
# Clear npm cache
npm cache clean --force
# Delete node_modules and try again
rm -rf node_modules
npm install
```

### Issue: Deploy fails
```bash
# Make sure you're on main branch
git branch
# Install gh-pages package
npm install --save-dev gh-pages
# Try deploying again
npm run deploy
```

### Issue: Site not showing updates
- GitHub Pages can take 5-10 minutes to update
- Try hard refresh: `Cmd + Shift + R`
- Check GitHub repository Settings > Pages for deployment status
